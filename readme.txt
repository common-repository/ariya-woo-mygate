The Ariya Woo MyGate plugin is developed to connect the website with MyGate payment gateway.

== Description ==

The Ariya Woo MyGate plugin allows the website to use the MyGate payment gateway. From admin panel the setting of the plugin can be updated.

== Installation ==

This plugin can be download and installed.

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings screen to configure the plugin
